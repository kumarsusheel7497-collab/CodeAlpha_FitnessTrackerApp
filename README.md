# 🏋️ Fitness Tracker App

A simple and user-friendly Fitness Tracker web application developed as part of the CodeAlpha App Development Internship.

## 📌 Features

- Add daily fitness activities
- Track exercise name
- Track workout duration
- Track calories burned
- View activity history
- Data stored using Local Storage
- Responsive and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage

## 📂 Project Structure

```
CodeAlpha_FitnessTrackerApp
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. Add your fitness activities.
4. View your activity history instantly.
